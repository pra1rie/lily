# lily

Very simple interpreter for a tiny language I'm making while learning rust.

This is a recreational project/challenge I started to make using rust more bearable.

## Rules

1. I can only work on it while drunk (3 beers minimum) or sleep deprived (20 hours w/o sleep minimum);
1. Goofy code is prioritized over "idiomatic" rust (which is plenty goofy tbh);
1. That's it.

## Alcohol Limitations

There's a Goldilocks zone of how much alcohol I can have before my rust starts turning into malbolge.
For me, personally, that is `2 < N < 6`. 5 beers is my limit where I can still more or less reason
about my abominable code, and coding in rust with less than 3 beers is no fun.

## Sleep Limitations

20 hours awake, with maybe 1 to 2 hours of sleep beforehand, is where I can still concentrate enough
on the code while I'm not hallucinating yet. Ideally should have lots of caffeine, or some beer
before working on the project.
